# HTTP prezentacija

Prezentacija se nalazi u ovom folderu kao staticki HTML/CSS/JS deck.

## Pokretanje lokalno

Iz korena repoa:

```powershell
python -m http.server 8088 --directory full_report/http_presentation
```

Zatim otvori:

```text
http://localhost:8088
```

## Navigacija

- `ArrowRight` / `Space`: sledeci slajd
- `ArrowLeft`: prethodni slajd
- klik na levi sadrzaj: direktan skok na slajd
